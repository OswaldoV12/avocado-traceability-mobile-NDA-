# Avocado Traceability Mobile (NDA Project)

## Overview

Mobile application developed in Android to support agricultural sample traceability from field collection to laboratory analysis.  
This project was built for a real organization and the source code cannot be shared due to confidentiality agreements (NDA).

## Project Context

The system was designed to centralize the management of avocado dry matter analysis, connecting field technicians, laboratory staff, and administrators through an integrated web and mobile solution.

The web platform handled orchard registration, user management, and report generation, while the mobile application focused on field data collection and laboratory validation.

## My Responsibilities

- Developed the mobile application using **Android (Java/Kotlin)**.
- Implemented authentication and role-based access control (Technician / Laboratory).
- Integrated REST APIs to synchronize data with a remote database hosted on Hostinger.
- Implemented offline data capture with deferred synchronization.
- Integrated **Google Maps API** to visualize KML files and collection points.
- Generated and scanned **QR codes** to ensure sample traceability.
- Implemented data capture features including weight registration, photo evidence, and sample identification.
- Integrated with portable thermal printers for QR label printing.
- Performed field testing and debugging.
